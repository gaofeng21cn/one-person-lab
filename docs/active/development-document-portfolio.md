## еҪ“еүҚз»“и®ә

OPL ејҖеҸ‘ж–ҮжЎЈзҺ°еңЁдёҚиғҪжҢүвҖңжҜҸд»Ҫж—§и®ЎеҲ’йғҪз»§з»ӯе®Ңж•ҙжү§иЎҢвҖқйҳ…иҜ»гҖӮеҪ“еүҚдё»зәҝе·Із»Ҹж”¶ж•ӣдёә framework-firstпјҡ

1. е…Ҳе®ҢжҲҗ OPL дҪңдёә stage-ledгҖҒд»Ҙ Agent executor дёәжңҖе°Ҹжү§иЎҢеҚ•дҪҚзҡ„е®Ңж•ҙжҷәиғҪдҪ“жЎҶжһ¶зҡ„ framework foundationгҖӮ
2. еҶҚи®© MAS/MAG/RCA иҝҒз§»дёә OPL-admitted domain agentsпјҢдҝқжҢҒ direct skill path дёҺ OPL-hosted path зӯүд»·гҖӮ
3. еҗҢжӯҘжҠҠж—§еҠҹиғҪйҖҗеқ—еҲҶеұӮпјҡframework-generic иғҪеҠӣдёҠж”¶еҲ° OPLпјҢdomain truth з•ҷеңЁ domainпјҢйҖҖеҪ№и·ҜзәҝеҸӘдҝқз•ҷдёәеҺҶеҸІиҜҠж–ӯгҖҒprovenanceгҖҒtombstone жҲ–иҙҹеҗ‘ guardгҖӮ
4. ж—§ Hermes-firstгҖҒGateway-eraгҖҒdirect-entryгҖҒlocal-managerгҖҒMDS-default зӯүи·ҜзәҝеңЁжӣҝд»ЈиҜҒжҚ®еӯҳеңЁеҗҺз«ӢеҚійҖҖеҪ№жё…зҗҶпјӣж—  active caller зҡ„жЁЎеқ—гҖҒжҺҘеҸЈе’ҢжөӢиҜ•зӣҙжҺҘеҲ йҷӨжҲ–иҝҒе…Ҙ tombstoneпјҢдёҚдҝқз•ҷе…је®№е…ҘеҸЈгҖӮ
5. жңҖеҗҺз”Ё App workbench е’Ңзңҹе®һ domain soak йӘҢиҜҒзӣ®ж ҮеҪўжҖҒпјҢиҖҢдёҚжҳҜз”Ёж—§и·Ҝеҫ„иҜҒжҳҺж—§и®ЎеҲ’гҖӮ

еӣ жӯӨпјҢејҖеҸ‘ж–ҮжЎЈзҡ„ж•ҙзҗҶеҺҹеҲҷжҳҜпјҡ**дҝқз•ҷжңүж•ҲеҶ…е®№пјҢеҗҲе№¶еҲ°еҪ“еүҚ ownerпјӣж—§ж–ҮжЎЈдёҚеҶҚдҪңдёәж•ҙд»Ҫеҫ…еҠһжү§иЎҢпјӣж—§и·Ҝзәҝдҝқз•ҷдёә provenanceгҖҒmigration reference жҲ– tombstoneгҖӮ**
